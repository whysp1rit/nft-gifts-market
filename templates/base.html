Инструкция по использованию
Для продавцов:
  1. Создайте сделку, указав ссылку на NFT подарок или username получателя
  2. Укажите сумму и валюту (звезды, рубли, гривны)
  3. Отправьте ссылку на сделку покупателю
  4. После оплаты переведите NFT подарок покупателю
  5. Для вывода средств пройдите верификацию аккаунта
Для покупателей:
  1. Перейдите по ссылке сделки от продавца
  2. Пополните баланс через администратора
  3. Оплатите сделку
  4. Дождитесь получения NFT подарка
